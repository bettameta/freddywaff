== Description ==

Provides one-click theme updates for all Graph Paper Press themes.

== Installation ==

1. Upload the whole `gpp-theme-updates` directory to the `/wp-content/plugins/` directory
1. Activate the 'GPP Theme Updates' plugin through the 'Plugins' menu in WordPress
1. With a Graph Paper Press theme activated, go to Appearance > Theme Updates and add your license key (available on your [member dashboard](http://graphpaperpress.com/members/member.php)) and follow the steps for automatically updating your GPP theme.
